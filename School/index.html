Overview

Validates input strings against a user-defined DFA. Users provide alphabet, states, final states, and transitions via a dynamic table; the app evaluates acceptance and surfaces results with clear alerts.

My contribution

  • Implemented table-driven DFA evaluation algorithm.
  • Built dynamic UI for transition input and validation feedback.
  • Improved UX with reset flow and alerts.
